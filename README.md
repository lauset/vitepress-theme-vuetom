## 简要说明

1. 基于 vitepress 打造的主题
2. 提供文档与博客两种风格的主题，更多的选择。

## 快速上手

- 拉取项目

```bash
git clone https://github.com/lauset/vitepress-theme-vuetom.git

cd vitepress-theme-vuetom

npm install
```

- 启动文档

```bash
npm run docs
```
- 构建样式

```bash
npm run build:theme
```

- 发布主题包
```bash
npm run pub
```

## 文档目录多语言

查看项目文件列表

```shell
$ crowdin:list
```

上传待翻译的源文件

```shell
$ crowdin:upload
```

查看预下载文件列表

```shell
$ crowdin:dryrun
```

下载en-US翻译文件

```shell
$ crowdin:us
```

下载zh-TW翻译文件

```shell
$ crowdin:tw
```
