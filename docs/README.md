## 开始操作

#### 安装依赖

```bash
$ npm i
```

#### 多语言

查看项目文件列表

```shell
$ crowdin:list
```

上传待翻译的源文件

```shell
$ crowdin:upload
```

查看预下载文件列表

```shell
$ crowdin:dryrun
```

下载en-US翻译文件

```shell
$ crowdin:us
```

下载zh-TW翻译文件

```shell
$ crowdin:tw
```

#### 本地开发

```shell
$ npm run dev
```

#### 线上环境

1. 打包

```shell
$ npm run build
```
2. 预览

```shell
$ npm run serve
```
